Summary of changes here!

A new Team Rocket Volcano structure has been added, along with a bunch of legendaries, Pecharunt, Terapagos, Meltan, Melmetal, Volcanion, Genesect, TypeNull, Munkidori, Okidogi and Fezandipiti.
